Welcome to the documentation of my REST API project!
===================================================

Project Description:

This project is a REST API application for storing and managing contacts. It provides the ability to create, update, delete, and retrieve contacts through HTTP requests.

Key Features:

1. Creating a new contact
2. Getting a list of all contacts
3. Getting a single contact by its identifier
4. Updating an existing contact
5. Deleting a contact

This documentation contains instructions on how to use each endpoint and request parameters.
